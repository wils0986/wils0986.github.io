Five HTML tags

Abbreviation
This tag is for an acronym or abbreviation, like “PIN”, “ATM”, etc. The title attribute contains the expanded version.
Blockquote
This tag is for a large, stand alone quote from another source.
Paragraphs
This tag is used to create a paragraph. The browser will add a space before and after the content of the paragraph (margins).
Headings
This tag is used to create a hierarchy on the page. The browser will style each heading differently, stating with the biggest size for h1 then h2 and so on.
Emphasis
This tag is used for a string of emphasized, slightly more important text. Screen readers will change their voice for this text.
